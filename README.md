# MarsGenix

This is a simple learning project built to understand backend development using Node.js and MongoDB.  
The main focus of this project is authentication and role-based access.

## Tech Stack
- Node.js
- Express
- MongoDB (Mongoose)
- JWT
- React (Vite)
- Plain CSS

## Features
- User signup and login
- JWT authentication
- User roles (customer, helper, admin)
- Protected routes
- Basic frontend for testing APIs
